# Logic Gates

**Part of:** [[DiscreteMathematics]]

Logic gates implement Boolean algebra operations and are the foundational discrete structures underlying cloud IAM policy evaluation, where policies are expressed as logical combinations of conditions.

## Applications in Multi-Cloud

- IAM policy condition evaluation (boolean expressions)
- Firewall rule matching
- Access control decision diagrams
