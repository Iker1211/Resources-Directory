# IAM Policies

**Part of:** [[DiscreteMathematics]]

IAM (Identity and Access Management) policies are logical expressions combining conditions with Boolean operators. Their evaluation is a direct application of discrete mathematics — specifically logic gates and predicate logic — in multi-cloud environments.

## Mathematical Foundations

- Boolean algebra for policy condition combination
- Predicate logic for quantified resource access statements
- Set theory for permission boundaries and scoping

## Applications

- AWS IAM policy evaluation engine
- Azure RBAC role assignments
- GCP IAM condition expressions
