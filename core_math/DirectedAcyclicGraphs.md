# Directed Acyclic Graphs (DAGs)

**Part of:** [[DiscreteMathematics]]

DAGs are the fundamental model for dependency resolution, workflow orchestration, and CI/CD pipelines in cloud infrastructure. They guarantee termination and enable topological sorting.

## Applications in Multi-Cloud

- CI/CD pipeline orchestration (GitLab CI, GitHub Actions)
- Infrastructure-as-Code dependency graphs (Terraform, Pulumi)
- Task scheduling and workflow engines (Argo Workflows, Airflow)

## Applications in Agentic AI

- Agent orchestration chains (DAG-based tool calling)
- Computation graphs in deep learning frameworks
