# Tensors

**Part of:** [[LinearAlgebra]]

Tensors are the fundamental data structure in modern machine learning frameworks (PyTorch, TensorFlow, JAX). They represent data across multiple dimensions — batch size, sequence length, embedding dimension, channels, etc.

## Applications in Agentic AI

- Neural network weight tensors
- Attention mechanism computations (Q, K, V tensors)
- Multi-head attention output concatenation

## Applications in Multi-Cloud

- Multi-dimensional monitoring metrics
- Tensorized resource utilization data
