# MultiCloudArchitecture

**Superclass:** [[MarketCompetency]]

Multi-cloud architecture relies on discrete mathematics for IAM policy evaluation (logic/boolean algebra), linear algebra for resource optimization and graph-based network topologies, and probabilistic systems for autoscaling and traffic prediction.

## Underpinned By

- [[DiscreteMathematics]] — IAM policy evaluation (logic gates, predicate logic)
- [[LinearAlgebra]] — Resource optimization, network flow analysis
- [[ProbabilisticSystems]] — Autoscaling telemetry, queueing theory
