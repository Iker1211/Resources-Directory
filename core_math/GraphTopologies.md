# Graph Topologies

**Part of:** [[DiscreteMathematics]]

Graph theory provides the mathematical language for describing network topologies in cloud infrastructure, including VPC peering, service meshes, and content delivery networks.

## Applications in Multi-Cloud

- Cloud network topology design (VPCs, subnets, transit gateways)
- Service mesh architecture (Istio, Linkerd)
- CDN edge node distribution
